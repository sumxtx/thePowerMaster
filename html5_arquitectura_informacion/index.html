Size of Datatypes

Depending on the platform the size may vary, as per example an integer in a 16 bit machine may be 16 bits, otherwise in a 32 bit machine it may be 32 bits.

Type Typical Byte Width Minimum Value Maximum Value
unsigned char 1 Byte 0 255
char 1 Byte -128 127
unsigned short int 2 Bytes 0 65535
short int 2 Bytes -32768 32767
unsigned int 4 Bytes 0 4294967295
int 4 Bytes -2147383648 2147483647
long int 4 bytes -2147483648 2147483647
unsigned long int 4 bytes 0 4294967295
float 4 Bytes 3.4x10-38 3.4x1038
double 8 Bytes 1.7x10-308 1.7x10308

Here is a code example on how to get the sizes and maximun values of data types

As the C/C++ Includes some predefined libraries that come in handy for this task 

          
            #include <bits/stdc++.h>

            int main()
            {
              std::cout << "Size of char: " << sizeof(char) << " Bytes."<< std::endl;
              std::cout << "Size of int: " << sizeof(int) << " Bytes."<< std::endl;
              return (0);
            }

Output: 

          
            Size of char: 1 Bytes.
            Size of int: 4 Bytes.
